2.  В `activity_main.xml` был размещен `ScrollView`, содержащий `LinearLayout` в качестве единственного дочернего элемента-контейнера.
3.  В `MainActivity.java` в цикле было программно создано 100 `TextView`. Каждому `TextView` было присвоено значение члена геометрической прогрессии (2^i), после чего он добавлялся в `LinearLayout`.

### 4. Контрольное задание - Интеграция в `HabitTracker`

**Задача:** Открыть основной проект (`HabitTracker`), создать "заглушку" с данными в репозитории и передать эти данные в `RecyclerView` через `LiveData` и `ViewModel`.

**Результат:**
При анализе текущей реализации проекта `HabitTracker` было установлено, что данное задание **уже было полностью выполнено** в ходе рефакторинга на предыдущей практической работе.

Текущая архитектура `HabitTracker` полностью соответствует требованиям задания:
1.  **"Заглушка" с данными** существует в классе `NetworkApi`, который используется репозиторием при первом запуске приложения.
2.  `HabitRepositoryImpl` загружает эти данные и передает их выше.
3.  `MainViewModel` получает данные и помещает их в `MutableLiveData<List<Habit>>`.
4.  `MainActivity` подписывается (`observe`) на это `LiveData` и, при получении обновленного списка, передает его в `HabitAdapter`, который в свою очередь обновляет `RecyclerView`.
